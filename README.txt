OSHIART STUDIO — WEBSITE PACKAGE
================================

Contents
  index.html      The website (one page). Keep it next to the images folder.
  images/         The 8 photos the page uses (web-optimized, geotagged).
  extra-photos/   5 spare web-ready photos, not currently on the page.

To preview on your computer
  Extract EVERYTHING from this zip, then double-click index.html.
  (If you move index.html without the images folder, photos will not show.)

To publish
  Upload index.html and the images/ folder together to your web host,
  keeping them side by side (e.g. both inside public_html).
